There's an uncomfortable gap in how we talk about AI in our work. "AI-generated" undersells the human judgement that shaped the result; pretending the AI was never there oversells it. Most of what I ship lives in between: I drove, the model helped.

So I've started calling it **AIssisted** — _AI_ + _assisted_, the second `i` doing double duty.

```
hand-made  ──  AIssisted  ──  AI-generated
   (you)        (you + AI)        (the AI)
```

Use it as a tag, a commit trailer, a footnote:

```
docs: rewrite onboarding guide (AIssisted)
```

It's honest in both directions. It says _I'm accountable for this_ — the typo, the bad call, the thing that breaks in prod is mine — while not claiming I typed every character myself. The AI was a tool, like an IDE or a linter, and tools don't need a co-author credit.

Steal if it's useful. In the meantime, I'll start using it as a tag here for transparency.
